# Клієнт-серверний додаток для сервісу обробки масивів

Даний проект є курсовою роботою на тему **«Написання клієнт-серверного додатку для сервісу обробки масивів - сортування і пошук»**. Робота виконана відповідно до вимог та стандартів **Київського університету інтелектуальної власності та права**.

## 📌 Опис проекту
Додаток реалізує розподілену систему за архітектурою «клієнт-сервер», де клієнт надсилає масиви даних на сервер через TCP-сокети, а сервер виконує їх обробку (сортування або пошук) та повертає результат із вимірюванням точного часу виконання операції.

### Реалізовані алгоритми та функції:
**Сортування:** * `Bubble Sort` (Сортування бульбашкою) — часова складність $O(N^2)$.
                  `Quick Sort` (Швидке сортування) — часова складність $O(N \log N)$.
* **Пошук:** * `Linear Search` (Лінійний пошук) — для невідсортованих даних.
               `Binary Search` (Бінарний пошук) — високоефективний пошук для відсортованих масивів.
*Багатопотоковість:** Сервер здатний обробляти запити від декількох клієнтів одночасно завдяки використанню потоків.

## 🛠 Технологічний стек
* **Мова програмування:** Python 3.x
* **Мережева взаємодія:** Бібліотека `socket` (TCP/IP сокети).
* **Формат даних:** JSON (для серіалізації та передачі масивів).
* **Аналіз продуктивності:** Модуль `time` (використання `perf_counter`).

## 🚀 Інструкція із запуску

Проект не потребує встановлення сторонніх бібліотек, оскільки використовує виключно стандартні інструменти Python.

 1. Запуск сервера
Відкрийте термінал і запустіть серверну частину:
```bash
python server.py
Сервер почне прослуховувати порт 5555 на локальній адресі 127.0.0.1.

2. Запуск клієнта
У новому вікні термінала (не закриваючи сервер) запустіть клієнтську програму:

Bash
python client.py
3. Використання
Після запуску клієнта з'явиться інтерактивне меню, де ви зможете:

Вказати розмір масиву для генерації.

Обрати алгоритм сортування або пошуку.

Отримати результат та час його обробки сервером.

📂 Структура репозиторію
server.py — містить логіку обробки запитів, реалізацію алгоритмів та серверний сокет.

client.py — реалізує інтерфейс користувача, генерацію випадкових чисел та клієнтський сокет.
